# Etsy Template Automation

A local pipeline that runs every morning at 8:00 AM and gives you:

1. 5 researched template opportunities per niche
2. A full design brief for each top idea
3. A mockup factory that turns one finished template PNG into 8 Etsy listing mockups
4. A listing writer that generates a 140-char Etsy title, 13 SEO tags, full description, and category

See the project root for setup and usage instructions.
